PASSOS PARA EXECUTAR:
	[1]: Execute a build com:
		make build
	
	[2]: Rode o programa com:
		make run
		
	[3]: Remova os arquivos compilados com:
		make cls
		
	[4]: Após executar o passo 1, a o rebuild/reboot pode ser feito com:
		make all
		OBS:. Caso o passo 3 seja executado, será necessário o passo 1 para que
		      o passo 4 volte a funcionar.
